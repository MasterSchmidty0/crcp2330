Cell Game
-----------------------------------------------------------------
Controls
UP: up arrow key
DOWN: down arrow key
LEFT: left arrow key
RIGHT: right arrow key
Q: quit/stop game
-----------------------------------------------------------------
Game Objectives
Eat other cells to win!
You can only eat cells that are smaller than you are.
Coming into contact with larger cells will cause you to be eaten.
To contact another cell, you must hit the cell exactly at its core.
-----------------------------------------------------------------
